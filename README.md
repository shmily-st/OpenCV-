基于 Python + OpenCV 开发的一站式图像处理工具箱，集成图像基础预处理、颜色阈值色块识别、几何图形与数字识别功能，并提供交互式参数调优工具，支持可视化结果输出与批量保存。

项目介绍

OPENPV 旨在简化计算机视觉领域的基础图像处理流程，为开发者 / 学习者提供开箱即用的工具集：

图像基础预处理：灰度转换、高斯模糊、直方图均衡化，生成多阶段效果对比图

颜色阈值色块识别：精准检测指定颜色范围的色块，输出色块数量、中心坐标、面积等信息

几何图形与数字识别：识别图像中的几何形状（圆形、矩形、三角形等）与数字，标注位置与类别

交互式参数调参工具：可视化调节颜色阈值、边缘检测、形态学操作、阈值处理等参数，实时预览效果

全流程支持自定义输入输出路径，自动创建输出目录，详细的错误提示与中断处理

## 使用方法

### 方式 1：交互式菜单运行

直接执行主程序，通过菜单选择功能，操作简单直观：

```
cd src 
python main.py
```

运行后将显示主菜单：

```
================================ OPENPV 主程序 ============================================================ 
1. 图像基础预处理
2. 颜色阈值色块识别 
3. 几何图形与数字识别 
4. 交互调参工具 
0. 退出 
============================================================ 
请输入选择 (0-4): 
```

#### 各功能操作说明

**1. 图像基础预处理**：输入图像路径（回车使用默认路径），程序自动完成灰度转换、高斯模糊、直方图均衡化，结果保存至 `output` 目录。

**2. 颜色阈值色块识别**：输入图像路径（回车使用默认测试图），检测图像中指定颜色的色块，输出数量、坐标、面积等信息，生成检测效果对比图。请将图片放在src下

**3. 几何图形与数字识别**：输入图像路径，识别图像中的几何形状与数字，标注结果并保存至输出目录。

**4. 交互调参工具**：选择调参模式（颜色 / 边缘 / 形态学 / 阈值），输入图像路径后，弹出交互式调参窗口，实时调节参数并预览效果。

### 方式 2：命令行参数辅助运行

支持通过命令行参数指定输入 / 输出路径、启用调试 / 调参模式：

```
# 指定自定义输出目录（--output输出结果目录（默认：output）） python main.py --output my_output_dir 
# 直接启用颜色调参模式（跳过菜单，直接进入调参--tune（可选值：color/edge/morphology/threshold））
python main.py --tune color --input images/color_test.jpg 
# 启用调试模式 python main.py 
```

#### 方式3：单独运行模块
```bash
cd src

# 图像预处理
python basic_preprocessing.py ../images/basic_test.jpg

# 颜色识别
python color_detection.py ../images/color_test.jpg

# 形状数字识别
python shape_number_recognition.py ../images/shape_number_test.jpg

```

## 实现思路

### 1. 图像基础预处理

```
输入图像 → 灰度转换 → 高斯模糊去噪 → 直方图均衡化 → 输出对比图
```

**关键算法：**
- `cv2.cvtColor()`: BGR转灰度/HSV
- `cv2.GaussianBlur()`: 高斯滤波去噪
- `cv2.equalizeHist()`: 直方图均衡化增强对比度

### 2. 颜色阈值色块识别

```
输入图像 → BGR转HSV → 颜色阈值分割 → 形态学去噪 → 轮廓查找 → 标注信息
```

**关键算法：**
- HSV颜色空间阈值分割（红色需处理0度跨越）
- 形态学开运算去噪、闭运算填补孔洞
- 轮廓面积筛选，计算中心点坐标

### 3. 几何图形与数字识别

#### 几何图形识别
```
轮廓提取→噪声过滤→特征分析→形状分类→可视化标注，按顶点数 + 圆形度的组合特征实现 8 类几何图形的分类
```

**形状分类核心（classify_shape）**：

计算轮廓矩，求解轮廓重心，为后续标注和特征分析做准备；

多边形逼近：用`cv2.approxPolyDP`通过轮廓周长的 4% 作为逼近精度，将不规则轮廓拟合为规则多边形，提取**顶点数**；

圆形度计算：通过公式`4π×面积/周长²`计算，圆形度越接近 1，形状越接近正圆，是区分圆形 / 椭圆 / 多边形的关键；

规则分类：按**顶点数**分基础多边形（3 三角、4 方 / 矩、5 五边、6 六边），4 顶点再通过**宽高比（0.9~1.1）** 区分正方形 / 矩形；顶点数＞6 的按**圆形度**分圆形（＞0.75）、椭圆（0.5~0.75）、多边形（≤0.5）

**数字识别**

```
轮廓层级提取→噪声过滤→双方法孔洞计数→规则化分类→可视化标注，按轮廓层级 + 孔洞数 + 形状特征的组合策略，采用 “孔洞数优先，形状特征辅助” 实现印刷体 0-9 的分类
```

 **交互调参运行核心流程**

命令行指定模式+图像路径 → 解析并校验参数 → 实例化调优类+初始化参数 → 读取图像+创建模式专属滑动条 → 实时监听滑动条拖动 → 回调更新参数+图像实时处理 → 双图拼接预览+参数显示 → 监听键盘事件（ESC退出/S保存）→ 保存参数到TXT+控制台打印最终参数 → 销毁窗口+程序退出

 **HSV 颜色阈值参考值**

| 通道        | 最小值（min） | 最大值（max） | 取值范围 |
| ----------- | ------------- | ------------- | -------- |
| H（色相）   | 0             | 179           | 0 ~ 179  |
| S（饱和度） | 100           | 255           | 0 ~ 255  |
| V（明度）   | 100           | 255           | 0 ~ 255  |

**形态学操作参数**

| 字段     | 取值                                                         |
| -------- | ------------------------------------------------------------ |
| 核大小   | 初始值 5，滑动条范围 0~212. 强制为奇数（偶数自动 + 1）3. 用于创建`(k_size, k_size)`矩形核 |
| 迭代次数 | 初始值 1，扩展后滑动条范围 1~52                              |

| 参数名称                          | 默认参考值           | 滑动条可调范围           | 特殊规则                                                  |
| --------------------------------- | -------------------- | ------------------------ | --------------------------------------------------------- |
| 结构元素（核）大小（kernel_size） | 3（3×3）             | 1 ~ 21（步长 1）         | 自动修正为**奇数**（OpenCV 强制要求），核越大操作效果越强 |
| 形态学操作类型（op_type）         | 开运算（MORPH_OPEN） | 0~4（映射 5 类常用操作） | 0 = 开运算、1 = 闭运算、2 = 腐蚀、3 = 膨胀、4 = 梯度      |
| 迭代次数（iterations）            | 1                    | 1 ~ 5                    | 迭代次数越多，操作效果越显著（易导致轮廓变形）            |

 **Canny 边缘检测参数参考值**

| 相关参数                   | 默认参考值 | 滑动条可调范围 | 特殊规则                          |
| -------------------------- | ---------- | -------------- | --------------------------------- |
| 高斯模糊核（blur_kernel）  | 5（5×5）   | 0 ~ 21         | 自动修正为**奇数**（OpenCV 要求） |
| Canny 低阈值（canny_low）  | 50         | 0 ~ 255        | 建议低于高阈值，经典比例 2:1/3:1  |
| Canny 高阈值（canny_high） | 150        | 0 ~ 255        | 建议高于低阈值，过滤虚假边缘      |
